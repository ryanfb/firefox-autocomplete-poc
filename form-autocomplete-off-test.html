Firefox Hidden Input Autocomplete Demo

Below is a form with hidden inputs that have a hardcoded HTML value of "test". When you submit the form, the values of the hidden inputs will be checked. If the value is "test", the value will be changed to the name of the input. Otherwise, an alert with the value will be shown. One of the form inputs has the autocomplete attribute set to "off".

This form differs from the main form only in also setting the autocomplete="off" attribute on the outer form element.

You can try to reproduce the behavior in this Firefox bug by clicking submit, then reloading the page, then clicking submit again. When you click submit after reloading, the value of the hidden input without autocomplete set to "off" will be the name of the input, when it should be the hardcoded value "test".
